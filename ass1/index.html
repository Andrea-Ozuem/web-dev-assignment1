INTRODUCTION TO CSS

CSS (Cascading Style Sheets) is a style sheet language. It is used to style HTML elements. CSS files always ends with .css CSS is always made with eference to an HTML document. This is done by linking your html file to your css file.

CSS RULESET

The basic structure of CSS consists of:

  1. Selectors
    This is the HTML element name at the start of thesyntax. It points to the elemnt to be styled.
  2. Declaration
    It specifies the style and the element property you want to style. It is simly key: value pair
  3. Property
    They are used within a declaration block to specify what property of the html elemnt you want to style .
  4. Property value
    This is the exact style you want to apply to the html property.
Other syntax rules include:
  • The declaration must be wrapped in curly braces ( {} )
  • Within each declaration, use a semicolon (;)to separate each declaration from the next one
  • Within each declaration, use a colon (:) to separate/ assign the property and the property value

CSS SELECTORS

There are different types of CSS selectors; id, class, element selectors, etc.

CSS element Selectors

This selects all HTML elements of the specified type. E.g 

        p {
          color: red
        }
We can select multiple elements we want with the same style definition by seperating each element with a comma(,). E.g 
          h,li {
            color:red;
          }

CSS class SELECTOR

There may be instances where you want to style a group of html elements. In that case, we use the class attribute.This selects the element(s) on the page with the same class. E.g 

          .underline {
            text-decoration: underline;
            }

CSS id SELECTOR

In cases where we need to style a specific elemnt, we use the id attribute. The id is unique within the entire html document. So, it is used to select just one unique element.
To select an element with an id, type the hash sign(#) followed by the id name. E.g 

            #my-id {
              color: red;
            }
NOTE: id is case sensitive

The CSS unversal selecotr(*) selects all HTML elemnts on the page

COMMON THINGS WE CAN DO WITH CSS

Change the Background color

            body {
              background-color: pink;
            }

Style the Body 

            body {
              color:grey;
              font-family: sans;
            }

Change the Heading

          h1 {
            text-color: black;
            font-weight: bold;
          }
